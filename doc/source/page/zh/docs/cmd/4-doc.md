`doc` 命令会在本地启动文档 HTTP 服务：

```go
pogo new site --doc
pogo doc --addr="0.0.0.0:9899"
```

执行 `new site` 后, `PoGo` 会释放文档数据到 `doc` 文件夹。 你可以使用 `doc` 命令编译和浏览文档内容。

`--addr` 设置 HTTP 服务的地址和端口，默认是 `0.0.0.0:9899`。
